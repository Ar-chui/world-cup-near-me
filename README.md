# world-cup-near-me
AI-enhanced event finder platform built for fans to discover fan zones, watch parties, and supporter gatherings for the World Cup near them.


# Project Overview

World Cup Near Me aggregates event information from multiple online sources and transforms unstructured event data into a searchable database.

Users will be able to:

Find nearby World Cup watch parties
Search events by team
Search events by match
View events on an interactive map
Discover supporter club gatherings and fan fests
Access original event sources

# Architecture

The platform combines:

Data Engineering pipelines for ingestion, transformation, and enrichment
AI-powered information extraction and classification
Geospatial search and event discovery
Full-stack web development
